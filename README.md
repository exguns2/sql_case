# SQL Case Study

Bu proje, Yıldız Teknik Üniversitesi - Elektronik ve Haberleşme Mühendisliği öğrencisi olarak i2i Systems staj süreci için hazırladığım SQL tabanlı veritabanı çalışmasıdır.

## 📁 Dosya İçeriği
- **CREATE.sql**: Veritabanı şemasını (TARIFFS, CUSTOMERS, MONTHLY_STATS tabloları) oluşturan scriptler.
- **SOLUTIONS.sql**: Ödev kapsamında istenen müşteri dağılımı, kota kullanım analizi ve ödeme durumu takibi gibi tüm fonksiyonel sorguları ve teknik açıklamaları içerir.

## 🛠️ Çalıştırma Talimatı
1. Oracle XE veritabanı üzerinde `CREATE.sql` dosyasını çalıştırarak tabloları oluşturun.
2. Gerekli veri setlerini (10.000 satırlık müşteri ve kullanım verisi) tablolara import edin.
3. Analiz sonuçlarını görüntülemek için `SOLUTIONS.sql` dosyasındaki sorguları sırasıyla çalıştırabilirsiniz.
